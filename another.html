Question-3: What is event delegation in the DOM?

Event delegation is a technique where a parent element handles events for its child elements using event bubbling.

Question-4: What is event bubbling in the DOM?

Event bubbling is a DOM mechanism where an event starts from the target element and propagates upward to its parent elements.

Question-5: How do you create, add, and remove elements using JavaScript?

Add Element: parent.appendChild(element). Remove Element: parent.removeChild(element)
